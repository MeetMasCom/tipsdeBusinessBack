Cargar las colecciones a la base de datos tips-business

PARA EL FRONT 
clonar el proyecto front 
cambiar las variables API-SERVER en el archivo src/app/utils/constants.ts 
ejecutar el proyecto con el comando ng serve

PARA EL BACK 
Clonar el proyecto back de tips business 
verificar las variables de coneccion a la base de datos en la ruta /src/database/conecction.ts 
en el archivo .env de la raiz del proyecto se encuentran todas las variables de coneccion 

ejecutar el proyecto back con el comando npm run dev
